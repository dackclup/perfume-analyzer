## Operational Insights for Future Rounds

- Pre-merge local verify catches what CI doesn't (full toolchain vs. CI's
  `npm run lint` surface check).
- Auto-delete works on Dependabot PRs (verified 5× in Phase 2).
- `@dependabot rebase` comment promotes pre-built rebase candidates (faster
  than waiting for Dependabot's scan cycle).
- Same-file PRs auto-rebase each other; different-file PRs don't.
- MCP authenticated fetches > unauthenticated curl (rate-limit avoidance).
- CI verification: read run ID + conclusion, NOT Pages-deploy success.
